## Overview

This repository will stay in sync with your deployed chats on [v0.app](https://v0.app).
Any changes you make to your deployed app will be automatically pushed to this repository from [v0.app](https://v0.app).

## Deployment

Your project is configured for deployment on Netlify.

### Environment Variables

Set the following environment variable in your Netlify dashboard (Site settings > Environment variables):

- `THINGSPEAK_READ_API_KEY`: Your ThingSpeak read API key

### Deploy Steps

1. Push this repository to GitHub or GitLab
2. Connect your repository to Netlify
3. Netlify will automatically build and deploy using the configuration in `netlify.toml`
